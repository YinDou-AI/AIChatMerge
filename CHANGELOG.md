# Changelog

## 1.0.0 - 2026-07-01

Initial public release.

### Added
- Multi-panel AI workspace with flexible layouts (up to 1×7)
- Send one prompt to multiple AI providers simultaneously
- Auto-merge answers from all providers into a single response
- Discussion mode for follow-up conversations
- Manual merge and auto-merge with post-merge discussion
- Markdown export for merged results
- Prompt library with variables and import/export
- Keyboard shortcut customization
- Chinese and English UI
- Local settings and prompt storage (no account required)
- Right-click context menu to send page content to AI
- Custom entry URL support for Claude
- Debug log export for issue reporting
