# AIChatMerge Documentation

This directory contains documentation and assets for the Chrome Web Store submission.

## Files

- `cws-store-listing.md` - Chrome Web Store 商店详情文案
- `index.html` - 隐私政策在线页面（GitHub Pages 用）

## GitHub Pages Setup

To enable GitHub Pages for the privacy policy:

1. Go to your repository Settings → Pages
2. Set Source to "Deploy from a branch"
3. Select branch: `main`, folder: `/docs`
4. Save

Your privacy policy will be available at:
`https://yindou-ai.github.io/AIChatMerge/`
